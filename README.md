## KHU EATS

- **Kyunghee University Side Project**
- Development Period : 2024.09.01 ~ 

## Project Introduction

**It is a community bulletin board that shares restaurants around the school.**

Only members were authorized to view the posts through **login sessions**. 

There are two types of bulletin boards in total. There are bulletin boards where you can freely write and share good restaurants. **You can post photos and write comments on the posts.**
